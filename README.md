# 🎬 Netflix User Behavior & Recommendation Analytics

## 📖 Project Overview

This project analyzes **Netflix user behavior and recommendation performance** using **Python (Pandas)** for data cleaning and exploratory data analysis (EDA), and **Power BI** for creating an interactive dashboard.

The goal of this project is to understand user viewing behavior, identify engagement patterns, evaluate recommendation performance, analyze user retention, and present actionable business insights through interactive visualizations.

---

# 🎯 Project Objectives

* Analyze user viewing behavior.
* Understand recommendation performance.
* Measure user engagement.
* Identify content consumption patterns.
* Analyze user retention.
* Build an interactive Power BI dashboard for business insights.

---

# 📂 Dataset

The dataset contains realistic Netflix streaming platform data with **26 columns**, including:

* User Information
* Subscription Plans
* Device Type
* Movie Details
* Watch Percentage
* Session Duration
* Recommendation Score
* User Retention Score
* Recommendation Click Status
* Viewing Hour
* Monthly Watch Count

# 💡 Key Business Insights

* Premium subscribers tend to have higher recommendation scores.
* Users with higher watch percentages are more likely to receive stronger recommendations.
* Longer viewing sessions are associated with higher retention scores.
* Favorite genres influence user engagement.
* Recently active users show better recommendation performance.
* Viewing behavior differs across countries and subscription plans.
